# 2. MVP 핵심 기능 3가지

## 2.1 Feature 1: AI 대화형 상담 엔진

### 목적
단순한 상품 비교를 넘어서 사용자의 상황을 이해하고 **컨설팅 수준의 조언**을 제공하는 AI 어시스턴트

### 핵심 가치
- **차별화**: 경쟁사는 자동 추천 → 우리는 대화형 컨설팅
- **신뢰도**: 기계적 추천 vs 맥락을 이해하는 조언자
- **체류시간**: 단순 비교(3분) → AI 대화(15-20분)
- **전환율**: 높은 만족도 → 승인율 상승

### 기술 스펙

**입력 방식**
- 텍스트 채팅 (필수)
- 음성 입력 (선택, Q2부터)
- 예시: "결혼 준비 중이고 전세자금 1억원 필요", "프리랜서인데 신용이 낮아요"

**AI 모델 선택**
- 추천: OpenAI GPT-4 API (한국 금융 파인튜닝) 또는 Google Gemini
- 대체안: Anthropic Claude API
- **NOT 추천**: 자체 구축 (초기 리소스 부족)

**핵심 기능**

| 기능 | 설명 | 우선순위 |
|------|------|---------|
| **맥락 이해** | 직업, 소득, 신용점수, 인생 목표 학습 | P0 |
| **상품 추천** | "당신에게 최적인 상품은 A입니다" 설명과 함께 | P0 |
| **승인율 안내** | "이 상품의 승인율은 87%입니다" 투명성 제시 | P0 |
| **질문 응답** | "내년 이직 예정인데 지금 대출받아도 되나?" | P1 |
| **비교 시뮬레이션** | 여러 상품의 금리, 한도, 상환액 자동 계산 | P1 |
| **음성 인식** | 텍스트 타이핑 대신 음성으로 말하기 | P2 (Q2) |

**데이터 입력 흐름**

```
사용자 입력
  ↓
AI가 필요한 정보 자동 추출
  ├─ 직업 분류 (공무원, 직장인, 프리랜서 등)
  ├─ 예상 소득 (연 3,000만원 ~ 1억원+)
  ├─ 신용점수 (앱에서 조회)
  └─ 대출 목적 (전세자금, 신사업 자금, 대환 등)
  ↓
최적 상품 3개 추천 + 설명
  ↓
사용자가 "더 자세히" 또는 "신청하기" 선택
```

**UI/UX 요구사항**

- 채팅 인터페이스 (카톡 스타일)
- AI 응답: 일반 텍스트 + **하이라이트 되는 주요 정보** (금리, 승인율, 수수료)
- 추천 상품 카드: 금융사명, 금리, 한도, 월 상환액, "신청하기" 버튼
- 응답 시간: **5초 이내** (느리면 AI라고 느껴지지 않음)

**성공 기준 (Q1 8주차)**
- 테스트 사용자 50명 중 **NPS 50 이상**
- AI 응답의 정확도 **90% 이상** (엔지니어 평가)
- 평균 채팅 시간 **15분 이상** (체류시간 지표)

## 2.2 Feature 2: 투명성 대시보드

### 목적
**업계 최초로 모든 정보를 공개**하여 신뢰도를 극대화하는 대시보드

### 핵심 가치
- **차별화**: 경쟁사는 수수료 숨김 → 우리는 모두 공개
- **신뢰도**: 투명성 = 금융의 가장 큰 이슈 해결
- **마케팅**: "투명성 대시보드" 자체가 PR 포인트
- **입소문**: 신뢰도 증가 → SNS 자연 확산

### 기술 스펙

**공개할 정보 (우선순위순)**

| 정보 | 형태 | 우선순위 |
|------|------|---------|
| **금융사별 수수료** | 숫자 + % 표시 | P0 |
| **추천 근거 설명** | 텍스트 + 3개 이유 | P0 |
| **실제 승인율** | 최근 30일 데이터 | P0 |
| **실시간 금리** | 각 금융사 현재 금리 | P1 |
| **검증된 리뷰** | 실제 대출 실행 고객만 | P1 |
| **경쟁사 비교** | 같은 상품 다른 곳 금리 | P2 |

**데이터 수집 방식**

- **수수료 정보**: 각 금융사와 직접 계약 (대출모집업 계약서에 명시)
- **승인율 데이터**: 자체 플랫폼에서 수집 (일일 갱신)
- **금리 정보**: 금융사 API 연동 (실시간)
- **리뷰 검증**: 블록체인 기반 (실제 대출 실행 내역과 매칭)

**UI 요구사항**

```
┌─────────────────────────────────┐
│  추천 상품: A은행 신용대출      │
├─────────────────────────────────┤
│ 📊 추천 이유                     │
│ ✓ 귀하의 신용점수(680) 구간에서 │
│   최저 금리 (7.8%)              │
│ ✓ 중도상환 수수료 0%            │
│ ✓ 최근 30일 실제 승인율 87%     │
├─────────────────────────────────┤
│ 💰 금융 정보                     │
│ • 금리: 7.8% ~ 11.2%            │
│ • 한도: 최대 5,000만원          │
│ • 상환기간: 3년 / 5년            │
├─────────────────────────────────┤
│ 🤝 당사 수수료                   │
│ • 대출금의 1.0%                 │
│ • A은행이 당사에 지급           │
│ • 귀하는 추가 비용 없음          │
├─────────────────────────────────┤
│ ⭐ 검증된 실제 리뷰             │
│ • 평점: 4.6/5.0 (2,847건)      │
│ • 실제 대출 실행 고객만 작성   │
│ • 블록체인 기반 검증            │
└─────────────────────────────────┘
```

**성공 기준 (Q1 8주차)**
- 제휴 금융사 5개 정보 연동 **정확성 99% 이상**
- 사용자 신뢰도 평가: "정보가 투명하다" **80% 이상 동의**

## 2.3 Feature 3: 직업군별 커뮤니티

### 목적
**온라인 전문가 세션 + 월간 오프라인 밋업**을 통해 신뢰 기반 네트워크 형성

### 핵심 가치
- **차별화**: 경쟁사는 커뮤니티 미보유 → 우리만의 강력한 차별화
- **경쟁 방어**: 따라할 수 없는 **네트워크 효과** 형성
- **CAC 절감**: 마케팅 비용 0으로 자연 확산 (MAU당 6,000원 → 최종 1,000원)
- **이탈율 감소**: 커뮤니티 있으면 이탈율 40% → 15%로 급감

### MVP 구현 범위

**Phase 1 (Q1 출시시점)**
- 커뮤니티 1개만 오픈: **공무원** (모수 400만명, 가장 큼)
  - 이유: 신용도 높음 → 초기 성과 가시화 쉬움
  - 이유: 직업 특성상 정보 공유 적극적

**Phase 2 (Q1 8주차 점검후)**
- 커뮤니티 추가 1개: **직장인** (모수 2,000만명)
- 함께 운영하며 프로세스 정리

**Phase 3 (Q2 오픈시)**
- 커뮤니티 2개 추가: **프리랜서**, **신혼부부**
- 최종 4개 커뮤니티로 Q2 종료

**Phase 4 (Q3-Q4)**
- 커뮤니티 6개 추가하여 총 10개 완성

### 기술 스펙

**커뮤니티 구성 요소**

| 요소 | 설명 | 우선순위 |
|------|------|---------|
| **게시판** | 공무원들의 대출 경험담 공유 | P0 |
| **Q&A** | 질문-답변 스레드 | P0 |
| **전문가 세션** | 주간 2회 라이브 (세무사, 재무설계사) | P1 |
| **월간 밋업** | 지역별 오프라인 모임 예약 시스템 | P2 |
| **리뷰** | "이 상품으로 얼마 절감했어요" 후기 | P1 |

**온라인 전문가 세션 (P1 - Q2부터)**

```
📅 매주 화요일 20:00, 목요일 19:00
👥 참여자: 공무원 커뮤니티원 50-500명
🎯 주제: "공무원 대환대출 성공 전략" 등
💬 형식: 라이브 스트리밍 (유튜브/인앱)
📝 진행: 공무원 선배 1명 + 세무사 1명
⏱️ 시간: 30분 진행 + 20분 Q&A
```

**월간 오프라인 밋업 (P2 - Q2부터)**

```
📍 장소: 주요 도시별 카페 또는 회의실
🗓️ 일정: 매월 2째 주 토요일 14:00
👥 참여자: 각 지역별 20-50명
🎯 목적: 신뢰 관계 형성, 입소문 마케팅
🏆 이벤트: 월간 베스트 후기 포상 (상금 50만원)
```
